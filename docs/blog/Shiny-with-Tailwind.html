Shiny with TailwindCSS

2022-03-27

A guide on how to use TailwindCSS to perform common styling tasks in Shiny applications.

Recently, I released the {shiny.tailwind} package which brings TailwindCSS classes to Shiny. This post is the first blog in a series of blogs that introduces TailwindCSS and shows how it can be used to quickly and easily design beautiful and responsive shiny applications

library(shiny.tailwind)


# Define UI for application that draws a histogram
ui <- div(
  # Load Tailwind CSS Just-in-time
  shiny.tailwind::use_tailwind(),
  div(className="w-full mx-auto flex flex-row gap-x-0",
    div(className="w-24 h-24 bg-red-300"),
    div(className="w-24 h-24 bg-green-300"),
    div(className="w-24 h-24 bg-blue-300")
  )
)
